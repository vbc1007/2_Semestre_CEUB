Estrutura de um documento HTML

A estrutura de um documento HTTML é definida pela declaração !DOCTYPE html, pelas tags html, head e body.

Essa estrutura é interpretada pelo navegador que analisa cada tag e rendereiza para o usuário.

A tag head é utilizada para configurações do navegador do documento HTML, as informações contidas dentro dessa tag são invisíveis para o usuário, com exceção do título que ficará visível na parte superior da página.

Já a tag body é utilizada para exibir toda parte visível para o usuário e nele teremos outras tags que também serão utilizadas pelo navegador.

Tag HTML, essa tag é a raiz que contém todas as outras tags do documento html, deliitando a totalidade do documento.

Por fim a instrução DOCTYPE html, que tem a finalidade de indicar ao navegador qual a versão do html que será utilizado. Ela deve vir na primeira linha do documento
