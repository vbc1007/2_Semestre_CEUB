O que é o Grid Layout

Article

O Grid Layout é bidmensional, permitindo que o desenvolveder possa separar o layout em linha e colunas, diferentemente do flexbox que é unidimensional, ou seja ordena os elementos em linha.

Article

Flexbox ou Grid Layout

O Flexbox e o Grid Layout podem ser usados em conjunto. Você pode aplicar o Flexbox a elementos dentro de um layout que esteja usando o Grid Layout, aproveitando o melhor dos dois mundos.

A escolha entre Flexbox e Grid Layout não se trata de um substituir o outro, mas sim de entender as necessidades específicas do seu layout e escolher a técnica que melhor atende a essas necessidades.
