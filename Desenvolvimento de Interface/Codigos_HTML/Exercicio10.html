Web Semântica

A web semântica é um conceito que visa organizar os dados na web, permitindo que máquinas os compreendam e interajam com ela (WEB). Para isso usamos várias tags semânticas que facilitaram o trabalho dos mecanismos de busca

Neste Exemplo estamos usanado a web semântica na prática, use um bloco de notas para acessar o documento html e ver como esta página foi construida
