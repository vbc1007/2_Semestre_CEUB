Exemplo 23 HTML CSS - Bordas e Alinhamento

Bordas e alinhamento são um importante recurso em CSS, eles são usados para definir o alinhamento e o posicionamento de elementos html.

Podemos utilizar diferentes propriedades para definir a margem(margin), ou preenchimento(padding) e a borda(border). Lembrando que a margin e o padding não tem cores, já a border oidera ter uma cor e diferentes formas de espessura e contorno.
