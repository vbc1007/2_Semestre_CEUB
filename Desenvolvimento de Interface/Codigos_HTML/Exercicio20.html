Cores e estilos de Fundo

Uso do Background com sistemas de cores RGB e RGBA

Sistema de cores RGB

O sistema de cores RGB representa um esquema de cores baseado nas cores R(Vermelho) G(Verde) B(Azul), cada parâmetro do rgb define a intensidade da cor e ela varia de 0 255. Pore exemplo rgb(255,99,71)

Sistema de cores RGBA

o sistema cores rgba são uma extensão dos valores rgb com canal alfa (a), que especifica a opacidade que pode variar de 0 a 1. Sendo o 0 totalmente transparente e o 1 totalmente opaco. Por exemplo rgba(35,99,71,0.3)
