Exemplo 19 HTML CSS - Vinculação Interna

Inline

A vinculação de arquivos CSS de forma interna pode ser realizada de várias formas diferentes. Uma delas é a vinculação inline, que é aquela onde a regra CSS é aplicada diretamente a um elemento HTML, usando o atributo style.
