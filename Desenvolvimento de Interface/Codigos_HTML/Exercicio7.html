Exemplo 7 - Uso das tags <p>, <hr>, <br>, <strong>, <em>

Este exemplo demonstra o uso de tags para formatação de texto e organização visual.

Descrição

As tags abordadas são:

  • <p>>: Define um parágrafo.
  • <hr>: Cria uma linha horizontal para separar conteúdos.
  • <br>: Insere uma quebra de linha.
  • <strong>: Define texto em negrito (com importância semântica).
  • <em>: Define texto em itálico (com ênfase semântica).
