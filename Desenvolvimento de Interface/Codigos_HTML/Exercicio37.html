Boostrap

O Bootstrap é um framework front-end que fornece componentes e ferramentas para facilitar a criação de sites responsivoso. Ele foi desenvolvido pelo Twiter e é muito util aqles que precisam de aplicar css a documentos html

Container

Container é uma classe fundamental do Bootstrap, usada para delimitar e alinhar o conteúdo na página, definindo limites para o layout e garantido o espaçamento adequado entre os elementos.

O seletor classe container pode ser aplicado de duas formas: container ou container-fluid. Sendo assim podemos dizer que o container quando aplicado delimita larguras fixas nas laterias do documento html. Já container-fluid ocupa, quando aplicado, 100% do layoyt da tela disponível
