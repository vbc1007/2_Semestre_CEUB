Exemplo 8 - Usos das tags <ol>, <ul> e <li>

As tags <ol>, <ul> e <li> representam listas em html, e são utilizadas para mais diversas finalidades. A tag <ol> representa uma lista ordenada ja a tag <ul> representa uma lista não ordenada.

Em ambas as listas para representar os elementos da lista usamos a tag <li>

Além disso, essas listas podem conter outras listas de forma alinhadas.

Etapas para Desenvolver um site(Lista Ordenada)

  1. Planejamento
    1. Definir o objetivo do site
    2. Indentificar o público alvo
    3. Criar um mapa do site(site map)
  2. Desing
    1. Criar wireframes e protótipos
    2. Definir a paleta de cores e tipografias
    3. Desenvolver layot no Figma ou Adobe XD
  3. Desenvolvimento
    1. Configurar o ambiente de desenvolvimento.
    2. Codificar o front-end(thm, css e javascript)
    3. Implementar o back-end, com linguagens orientadas a objeto(C#, Java, PHP, Python)
  4. Testes
    1. Testar a responsividade em diferente dispositivos
    2. Realizar testes de usabilidade
    3. Corrigir bugs identificados
  5. Publicação
    1. Escolher um provedor de hospedagem
    2. Configurar o domínio SSL
    3. Fazer deploy do site.
