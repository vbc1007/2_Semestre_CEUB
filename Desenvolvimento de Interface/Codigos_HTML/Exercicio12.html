Exemplo 12 HTML - Formulários

Os formulários em HTML são um dos principais recursos para coleta de dados, eles devem ser simples e altamente funcionais, atendendo sempre as necessidades dos usuários.

Para trabalhar com formulários em html, possuimos uma serie de componentes que vão facilitar o processor de interação com o usuário. Todos os formulários começam necessariamente com a tag <form>, essa tag indica para o navegador que página html possui formulários que pode enviar dados.

Além da tag <form> temos também que usar os atributos action (que define o local = url = em que os dados recolhidos/coletados devem ser enviados.), e o method que define qual mètodo http para enviar os dados GET ou POST
