Exemplo 38 Bootstrap - Cores Contextuais

O Bootstrap utiliza um sistema de cores Contextuais para facilitar a marcação e a estilização de elementos com base em seu significado ou proposito. Essas classes de cores são usadas para dar siginificado visual aos elemntos html. Geralmente utilizamos essas cores em botões, alertas, barras de navegação e outros elementos que são utilizados em interfaces gráficas.
